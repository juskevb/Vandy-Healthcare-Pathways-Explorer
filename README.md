# Vandy Healthcare Pathways Explorer

A career exploration website for Vanderbilt students interested in the non-clinical side of healthcare: six tracks, each with a plain-language field overview, a 4-task virtual work-sample project with sample answers, and Vanderbilt-specific next steps.

**Live site:** https://juskevb.github.io/Vandy-Healthcare-Pathways-Explorer/

The whole site is the single `index.html` file (no build step, no dependencies). Published with GitHub Pages (Settings > Pages > Deploy from a branch > main, root). Student progress saves in the student's own browser; nothing is collected or transmitted.
